# 🛒 Modern Shopping Cart

Продуманное до мелочей приложение для шопинга с упором на **архитектуру, UX и стабильность**. Это не просто корзина, а полноценный инженерный кейс, включающий тестирование, кастомные хуки и сложную логику состояний.

🔗 \*\*[Посмотреть живое демо (Vercel)](https://shopping-cart-pearl-zeta.vercel.app/)

---

## 💎 Ключевые особенности (Highlights)

- **🛡️ Надежность:** Покрыто Unit-тестами (Vitest + React Testing Library) для бизнес-логики корзины и UI-компонентов.
- **⚡ Перформанс:** Реализованы **Skeleton Screens** для предотвращения резких скачков контента (Layout Shift) во время загрузки.
- **💾 Настойчивость:** Полная синхронизация с `localStorage` — корзина не пропадает после перезагрузки страницы.
- **🎨 Эстетика:** Мягкие анимации (Fade-in/down), адаптивный дизайн (Mobile First) и кастомные уведомления (Toasts) об операциях.
- **🛑 Обработка ошибок:** Кастомный экран ошибок и пустых состояний (Empty States) для улучшения пользовательского опыта.
- **⚙️ Инженерная культура:** Чистая история коммитов по стандарту **Conventional Commits** и работа через **Pull Requests**.

## 📐 Архитектурные решения

- **`useFetch` Hook:** Абстракция над логикой запросов с использованием `AbortController` для предотвращения утечек памяти при размонтировании компонентов.
- **Cart Logic:** Централизованное управление состоянием через Context API с оптимизацией количества ререндеров.
- **Adaptive Layout:** Использование динамических единиц измерения (`dvh`) и гибкой Flexbox-верстки для идеального отображения на мобильных устройствах без лишнего скролла.

## 🧪 Тестирование

Проект следует принципам надежного кода. Запуск тестов:

```bash
npm run test

```

- **Unit-tests:** Проверка корректности добавления/удаления товаров и подсчета общей суммы.
- **Component-tests:** Имитация действий пользователя (Click, Input) для проверки отзывчивости интерфейса.

## 🚀 Как запустить локально

1. Клонировать репозиторий:

```bash
git clone https://github.com/SAGOMADGE/Shopping-Cart.git

```

2. Установить зависимости:

```bash
npm install

```

3. Запустить сервер разработки:

```bash
npm run dev

```
